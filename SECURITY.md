# Security Policy

## Reporting A Vulnerability

Please report security issues privately by emailing security@sawfwair.com.

Do not open a public issue for leaked credentials, provider account access,
remote execution bypasses, or cleanup failures that could leave paid resources
running. Include the affected plugin, the command you ran, and the smallest
non-secret reproduction you can share.

## Supported Versions

This repository is pre-1.0. Security fixes apply to the current `main` branch
until tagged releases are introduced.
