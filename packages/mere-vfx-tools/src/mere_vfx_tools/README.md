# mere_vfx_tools package

The package contains the contract-compliant `mere-vfx-tools` CLI. `cli.py`
defines planning, durable run manifests, local execution, artifact hashing,
resumability, and the current shot-oriented VFX workflows. Native model
inference is invoked through the user-selected `mere.run` command; deterministic
media assembly uses the user-selected `ffmpeg` command.
